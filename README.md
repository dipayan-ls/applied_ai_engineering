# Applied AI Engineering

> The definitive textbook for engineers who build, deploy, and lead production-grade AI systems.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Serve locally with live reload
mkdocs serve

# Build static site
mkdocs build
```

Navigate to `http://localhost:8000` after running `mkdocs serve`.
