## Foundational Papers

| Year | Title | Authors | Key Contribution |
|---|---|---|---|
| 1943 | A Logical Calculus of Ideas Immanent in Nervous Activity | McCulloch & Pitts | First mathematical model of a neuron |
| 1950 | Computing Machinery and Intelligence | Alan Turing | The Turing Test; framework for machine intelligence |
| 1958 | The Perceptron | Rosenblatt | First trainable neural network |
| 1986 | Learning Representations by Back-propagating Errors | Rumelhart, Hinton, Williams | Backpropagation algorithm |
| 1989 | Multilayer Feedforward Networks are Universal Approximators | Hornik et al. | Universal approximation theorem |
| 1997 | Long Short-Term Memory | Hochreiter & Schmidhuber | LSTM — solved vanishing gradients in RNNs |
| 1998 | Gradient-Based Learning Applied to Document Recognition | LeCun et al. | LeNet; convolutional networks for images |
| 2006 | A Fast Learning Algorithm for Deep Belief Nets | Hinton, Osindero, Teh | Deep learning renaissance |
