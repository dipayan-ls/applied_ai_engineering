## Course Philosophy

!!! quote "The Applied AI Engineering Manifesto"
    **Understand every abstraction before using it.**
    Build concepts from scratch before reaching for a library.
    Focus relentlessly on production engineering — not demos.
    Learn the *why*, not only the *how*.

This is not a collection of tutorials. This is a complete engineering textbook that treats AI with the same rigour applied to systems engineering, distributed computing, or compiler design.

## How to Use This Textbook

=== "Self-Study"
    Read sequentially from Volume 1 through Volume 10.
    Complete every exercise and lab before moving to the next chapter.
    The prerequisite graph is strictly enforced — resist the temptation to skip.

=== "Reference"
    Each chapter is self-contained with a complete index, glossary, and cross-references.
    Use the search bar to navigate directly to any concept, algorithm, or technique.

=== "Bootcamp"
    If you have a strong background, start at the volume matching your current level.
    Check prerequisites in the chapter header to confirm readiness.

=== "Course Instructors"
    Each chapter includes learning objectives, exercises, quizzes, and labs.
    See the [Instructor Guide](about.md) for syllabi suggestions.
